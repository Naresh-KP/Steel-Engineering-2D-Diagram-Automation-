# 🏗️ Steel Engineering 2D Diagram Automation

An automated drafting solution that delivers **Instant Client-Side Vector Renders** by utilizing lightweight HTML5 Canvas routines to generate precise, scale-compliant 2D structural drawings directly in any standard browser viewport—bypassing the need for heavy CAD software installations.<br/><br/>
Engineered with a **Parametric Data Mapping Matrix** that parses complex multi-variable engineering inputs—including section heights, flange/web dimensions, bolt pitch configurations, and gusset plate alignments—and automatically re-computes them into pixel-perfect layout schemes in real time.<br/><br/>
Features **Smart Drafting Geometry Compliance** with built-in logic matrices that automatically interpret engineering detailing conventions to render solid vector paths for visible profile edges, dashed configurations for hidden geometry, and real-time dynamic dimension line strings.<br/><br/>
Built on a **Zero-Footprint Desktop & Web Pipeline** with absolutely zero dependencies or server-side rendering overhead, ensuring maximum execution speed whether deployed live via GitHub Pages or compiled into a standalone offline Windows executable (`.exe`).

## 📐 Engineering Requirements & Design Matrix

This automation workspace parses multi-variable structural inputs to render standardized component sections. The system logic is architected to process the following technical requirements:

### 🔹 Core Input Parameters & Dimensional Bounds
* **Section Specifications:** Configurable height, flange width, flange thickness, and web thickness parameters matching industry standard steel shapes (such as I-Beams, Wide Flange Sections, and Channels).
* **Connection Logic:** Interactive bolt placement spacing, pitch distance configurations, and structural gusset plate edge alignments.
* **Scale Resolution:** Dynamic millimetric scaling variables that automatically re-compute pixel grid distributions for standard drawing limits.

### 🔹 Blueprint Rendering & Drafting Compliance
* **Standard Line Types:** Automatic assignment of solid lines for visible profile edges and dashed alignments for underlying hidden geometry details.
* **Dynamic Dimensioning Line Strings:** Real-time generation of calculation boundaries, measurement text markers, and extension ticks conforming to typical engineering detailing rules.
* **Self-Contained Rendering Runtime:** Employs math matrices to map coordinate transformations within browser boundaries, removing server-side layout calculation delays.
